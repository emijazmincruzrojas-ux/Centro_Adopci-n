# 🐾 Centro de Adopción

Aplicación web desarrollada con Flask y MariaDB para gestionar un centro de adopción de mascotas.

---

## 🚀 Características

- Registro de mascotas
- Catálogo de adopción
- Historial de adopciones
- Confirmación de adopción

---

## 🛠️ Tecnologías utilizadas

- Python
- Flask
- MariaDB
- HTML / CSS

---

## 📦 Instalación

git clone https://github.com/emijazmincruzrojas-ux/Centro_Adopci-n.git

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

---

## ⚙️ Configuración

export DB_PASSWORD=tu_contraseña

---

## ▶️ Ejecutar

python main.py

Abrir en navegador:
http://127.0.0.1:5000
